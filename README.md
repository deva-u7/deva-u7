## What I work on

Backend systems in Java and Kotlin — mostly Spring Boot and Quarkus. Production stack covers messaging (Kafka, SNS/SQS), AWS, multiple databases, BPMN orchestration with Camunda, and the usual DevOps surface.

On the side I'm exploring AI engineering — local LLMs with Ollama, RAG pipelines, agent patterns. These are non-production experiments while I build up to a real product.
